# Kth Largest Element

**Difficulty:** Easy

## Problem Description

Given an array of integers `nums` and an integer `k`, return the kth largest element. Use a heap/priority queue or sort.

## Examples

**Example 1:**
- Input: `nums = [3, 1, 5, 2, 4], k = 2`
- Output: `4`

**Example 2:**
- Input: `nums = [1, 1, 1], k = 1`
- Output: `1`
