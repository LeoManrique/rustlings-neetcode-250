# Merge Overlapping Intervals

**Difficulty:** Easy

## Problem Description

Given a list of intervals `[start, end]`, merge all overlapping intervals and return the result sorted.

## Examples

**Example 1:**
- Input: `[[1,3],[2,6],[8,10]]`
- Output: `[[1,6],[8,10]]`

**Example 2:**
- Input: `[[1,4],[4,5]]`
- Output: `[[1,5]]`
