# Count Connected Components

**Difficulty:** Easy

## Problem Description

Given `n` nodes labeled `0` to `n-1` and a list of undirected edges, return the number of connected components.

## Examples

**Example 1:**
- Input: `n = 5, edges = [[0,1],[1,2],[3,4]]`
- Output: `2`

**Example 2:**
- Input: `n = 3, edges = []`
- Output: `3`
