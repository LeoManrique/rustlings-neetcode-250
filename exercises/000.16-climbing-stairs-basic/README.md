# Climbing Stairs

**Difficulty:** Easy

## Problem Description

You are climbing a staircase with `n` steps. Each time you can climb 1 or 2 steps. Return the number of distinct ways to reach the top.

## Examples

**Example 1:**
- Input: `n = 3`
- Output: `3` (1+1+1, 1+2, 2+1)

**Example 2:**
- Input: `n = 1`
- Output: `1`
