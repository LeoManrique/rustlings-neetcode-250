# Detect Cycle in Graph

**Difficulty:** Easy

## Problem Description

Given `n` nodes (0-indexed) and directed edges, determine if the graph contains a cycle.

## Examples

**Example 1:**
- Input: `n = 3, edges = [[0,1],[1,2],[2,0]]`
- Output: `true`

**Example 2:**
- Input: `n = 3, edges = [[0,1],[1,2]]`
- Output: `false`
