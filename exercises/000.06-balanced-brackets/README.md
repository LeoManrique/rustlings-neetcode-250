# Balanced Brackets

**Difficulty:** Easy

## Problem Description

Given a string `s` containing only `(` and `)`, return `true` if the parentheses are balanced.

## Examples

**Example 1:**
- Input: `s = "(())"`
- Output: `true`

**Example 2:**
- Input: `s = ")("`
- Output: `false`

**Example 3:**
- Input: `s = "((())())"`
- Output: `true`
