# Reverse String with Stack

**Difficulty:** Easy

## Problem Description

Given a string `s`, reverse it using a stack. Push all characters onto a stack, then pop them off to build the reversed string.

## Examples

**Example 1:**
- Input: `s = "hello"`
- Output: `"olleh"`

**Example 2:**
- Input: `s = "ab"`
- Output: `"ba"`
