# Binary Search

**Difficulty:** Easy

## Problem Description

Given a sorted array of integers `nums` and a `target`, return the index of `target` if found, or `-1` if not found. You must use binary search.

## Examples

**Example 1:**
- Input: `nums = [1, 3, 5, 7, 9], target = 5`
- Output: `2`

**Example 2:**
- Input: `nums = [1, 3, 5, 7, 9], target = 4`
- Output: `-1`
