# Array Sum

**Difficulty:** Easy

## Problem Description

Given an array of integers `nums`, return the sum of all elements.

## Examples

**Example 1:**
- Input: `nums = [1, 2, 3]`
- Output: `6`

**Example 2:**
- Input: `nums = [-1, 0, 1]`
- Output: `0`

**Example 3:**
- Input: `nums = []`
- Output: `0`
