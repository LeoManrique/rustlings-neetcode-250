# Binary Tree Height

**Difficulty:** Easy

## Problem Description

Given the root of a binary tree, return its height (number of edges on the longest path from root to a leaf). An empty tree has height `-1`, a single node has height `0`.

## Examples

**Example 1:**
- Input: `root = [1, 2, 3]`
- Output: `1`

**Example 2:**
- Input: `root = [1, 2, null, 4]`
- Output: `2`
