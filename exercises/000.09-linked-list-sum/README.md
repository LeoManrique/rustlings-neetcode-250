# Linked List Sum

**Difficulty:** Easy

## Problem Description

Given the head of a singly linked list of integers, return the sum of all node values.

Use the provided `ListNode` definition.

## Examples

**Example 1:**
- Input: `1 -> 2 -> 3`
- Output: `6`

**Example 2:**
- Input: `empty list`
- Output: `0`
