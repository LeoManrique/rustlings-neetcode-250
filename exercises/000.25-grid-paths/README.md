# Grid Unique Paths

**Difficulty:** Easy

## Problem Description

Given an `m x n` grid, find the number of unique paths from the top-left to the bottom-right corner. You can only move right or down.

## Examples

**Example 1:**
- Input: `m = 3, n = 2`
- Output: `3`

**Example 2:**
- Input: `m = 1, n = 1`
- Output: `1`
