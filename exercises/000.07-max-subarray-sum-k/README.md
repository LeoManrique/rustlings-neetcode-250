# Max Subarray Sum of Size K

**Difficulty:** Easy

## Problem Description

Given an array of integers `nums` and an integer `k`, find the maximum sum of any contiguous subarray of size `k`.

## Examples

**Example 1:**
- Input: `nums = [1, 3, 2, 6, 1], k = 2`
- Output: `8` (subarray [2, 6])

**Example 2:**
- Input: `nums = [1, 1, 1, 1], k = 2`
- Output: `2`
