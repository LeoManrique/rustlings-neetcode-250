# FizzBuzz

**Difficulty:** Easy

## Problem Description

Given an integer `n`, return a vector of strings from `1` to `n` where: multiples of 3 are `"Fizz"`, multiples of 5 are `"Buzz"`, multiples of both are `"FizzBuzz"`, otherwise the number as a string.

## Examples

**Example 1:**
- Input: `n = 5`
- Output: `["1", "2", "Fizz", "4", "Buzz"]`
